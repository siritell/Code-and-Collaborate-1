# TrendWear – Agile UX & Frontend Project

🔗 Live site: https://siritell.github.io/Code-and-Collaborate-1/shopping-bag.html

## 📌 Overview
TrendWear is a mobile-first landing page for a fictional clothing brand. The project focuses on creating a simple shopping experience where users can browse products, view sales, and add items to a favorites list.

This project was built as part of a Hyper Island course with a strong focus on Agile workflows, UX design, and collaborative frontend development.

---

## 🎯 Features
- Browse clothing items on a landing page  
- Filter products (Sale / New Arrivals)  
- Add items to a favorites list  
- Clear visibility of discounted items  

---

## ⚙️ Tech Stack
HTML • CSS • JavaScript • Figma • Git / GitHub  

---

## 🔄 Process
We followed an Agile workflow with weekly sprints, daily stand-ups, and a Kanban board for task management.

- UX Research: Competitive analysis + user interviews (15 participants)  
- Design: Crazy 8 ideation, wireframes, and Figma prototyping  
- Testing: 2 rounds of usability testing (prototype + live site)  
- Iteration: Improvements based on user feedback  

---

## 🧪 Key Improvements from Testing
- Moved navigation from bottom to header for better usability  
- Improved visibility of sale items with stronger visual styling  
- Added user feedback when adding items to favorites  

---

## 💡 Learnings
- Working in Agile sprints with a team  
- Translating UX research into functional UI  
- Iterating based on real user testing  
- Structuring frontend code collaboratively with Git

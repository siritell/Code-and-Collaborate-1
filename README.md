# TrendWear

TrendWear is a mobile-first fashion shopping web app focused on clean design, fast product discovery, and intuitive interactions.
The project aims to reduce visual clutter and improve usability by prioritizing essential product information, clear navigation, and a seamless favorites experience.

# Key Features

✅ Product listing with large images and clear pricing.
✅ Product detail page with image slider.
✅ Add to cart functionality.
✅ Favorites (wishlist) with instant feedback.
✅ Toast notifications for user actions.
✅ Accordion sections for size, material, and description.
✅ Mobile-first, minimal UI design.
